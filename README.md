Local scene conversion:

```bash
npm install
npm run convert:ply -- Scenes
```

Useful variants:

```bash
npm run convert:ply -- Scenes --force
npm run convert:ply -- Scenes --recursive
npm run convert:ply -- Scenes/AC_day.ply
```
